# Commit Convention

- **[FEAT]** : Implement a new feature  
- **[MOD]** : Modify existing code or internal files  
- **[ADD]** : Add supplementary code, libraries, or new files  
- **[CHORE]** : Minor tasks such as version updates, package restructuring, or renaming variables/types  
- **[DEL]** : Delete unnecessary code or files  
- **[FIX]** : Resolve bugs or errors  
- **[MERGE]** : Merge changes from another branch  
- **[MOVE]** : Move files or code within the project  
- **[RENAME]** : Rename files  
- **[REFACTOR]** : Refactor code without changing functionality  
- **[DOCS]** : Update documentation such as README or Wiki  
- **[DEPLOY]** : Deployment-related tasks
